tilt

Github

tilt is a React Native app with the use of a 3rd party API called Rawg.io that allows users to search for games, leave reviews, and add comments! Users can also add games to different lists on their profile such as 'Favorites,Played, and what youre keeping an eye on' from a database collection of over 350,000 games to choose from. Postman played a huge part in the production of this app. This app was made with React Native, bootstrap, node.js, Mongo.DB, express, CSS, javascript, mongoose. This app is deployed on Heroku.

DnD Character Pub

Github

DnD Character pub is a Full Stack CRUD app that allows users to customize their own characters. Users are able to leave reviews and comments on other users characters. Each character has a average rating score based off of what other users think. This project incorporates one of my favorite hobbies and i enjoyed building this from scratch. I plan to add a lot more functionality which is listed in the Readme section on my github. Made with express, node.js, Google Oauth, CSS. This app is deployed on Heroku.

DayByDay

Github

DayByDay is a Full Stack CRD app that allows users to track their daily life mood. The Homepage is a collection of the users data and depending how the user rated their day, the card will be either green, yellow or red. The purpose is to let the users see how theyre doing mental health wise as a whole. This app is meant for personal use and has a sign in requirement. Made with Django, Python, Bootstrap, CSS. This app is deployed on Heroku.

Slot Machine Game

Github

This app is an interactive minimalist slot machine with credits balance system. With 7 different winning combinations, Users can win the jackpot worth 5000 credits or lose the chance game and lose up to 5000 credits! This game features a light and dark mode to further enhance the user experience. Made with Javascript, bootstrap, and CSS. This app is deployed on Netlify.
